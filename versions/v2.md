# v2 — "Rock & Rumble"

**Released:** March 24, 2026
**URL:** [ribbon.obfusco.us/v2](https://ribbon.obfusco.us/v2)

Major overhaul with 3-oscillator engine, polyphony, arpeggiator, 3D visuals, and shareable QR presets.

## Sound Engine
- Third oscillator — full 3-osc engine with independent waveform, detune, and mix
- Bitcrush/Crunch effect — AudioWorklet-based bit depth + sample rate reduction with slapback delay
- Polyphonic voice pool — up to 8 simultaneous voices with voice stealing
- Filter controls — cutoff frequency (20-20kHz) and resonance (0-25)
- Glide/portamento — adjustable pitch slide speed

## Play Modes
- Arpeggiator with adjustable BPM (40-300), works with poly and hold
- Mono/Poly toggle — hardware-style rocker switch
- Hold mode — sustains notes; in arp+poly+hold, tap ribbon to build note sequences
- Arp trigger markers — visual lines on ribbon showing active arp notes

## Controls
- Hardware rocker switches — 3D physical-style toggles for Play/Arp and Mono/Poly
- DJ volume fader — vertical mixer-style fader
- Per-section shake bolts — randomize individual oscillators or general section
- Multi-select scales — combine scales for hybrid note sets
- Shake combo system — rapid triggers increase intensity

## Visuals
- 3D sphere visualizer — Three.js wireframe spheres (one per oscillator), audio-reactive, zoomable
- Staff notation — scrolling neon musical staff with flag-wave animation
- Industrial panel skin — beveled panels, brushed steel, retro console aesthetic
- Per-OSC colored slider handles with subtle hue shifts

## Mobile
- Accelerometer shake — DeviceMotionEvent for iOS/Android
- Floating lightning bolt shake button
- Compact layout — ribbon above controls, two-row switches, horizontal faders
- Audio fixes — AudioWorklet secure context guard, gesture-based AudioContext resume

## Sharing
- QR code presets — all settings encoded in URL hash, multi-colored gradient QR codes
- Preset names embedded in QR code visual and URL
- Downloadable QR PNG with preset name

## Easter Eggs
- Hidden double harmonic scale — ~5% chance on shake unlocks Byzantine/Arabic scale
- Hidden 5th octave

## Infrastructure
- CI/CD — auto-deploy to S3/CloudFront on push to main
- Dev deploys to ribbon-dev.obfusco.us on push to nmj/* branches
- MIDI controller support via Web MIDI API
- Keyboard input mode (A-L keys)
