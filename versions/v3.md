**Design direction:** Glitch/datamosh/chromatic aberration aesthetic

## Planned Features

### Crypto Integration
- NFT/EFT gain from usage — transparent, simple implementation
- POAP support (proof of attendance protocol)

### Ambient Play
- Mild ambient autoplay mode — gentle ribbon taps with reverb/delay
- Slowly evolving controls for generative ambient soundscapes
- User-togglable via ambient icon

### Visual Overhaul
- Glitch/datamosh/chromatic aberration aesthetic (inspired by v3 design direction screenshots)
- New version theme identity

### Interactive 3D
- Grab/drag oscillator spheres to change settings and/or create sound

- Additional scales: Phrygian dominant, Hirajoshi, Whole tone, Prometheus, Hungarian minor, Enigmatic, Lydian dominant
- Step sequencer functionality

### UX

- Camera input mode for ribbon control
- Rotary knob components (replace range sliders)
